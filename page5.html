Stylesheets

External style sheets are the most preffered ay of styling HTML documents. Use one files allows a developer to be more efficient. When creating the CSS style sheets the file must be saved with a .css extension. Each HTML page must include a reference to the external style sheet. This is accomplished by using the link element.

The rel attribute, mandatory, specifies the relationship between the HTML page and the file it is linked to. The value should be stylesheet when linking to a CSS file. type:This specifies the type of the document being linked to. href:This specifies the path or location to the CSS file. Often the CSS file is placed within a folder called styles.

Internal stylesheets are applied by placing the CSS rules inside of a style element, which is nested within the head element. The style element has opening and closing tags.This method should only be used if a site is being built with one web page.

Inline stylesheets are applied to HTML page by adding the style attribute to the element you want to style. The CSS rules will only apply to that one element
