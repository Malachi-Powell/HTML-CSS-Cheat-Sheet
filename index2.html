Git Commands...

  • cd (change directory):Changes the directory or folder
  • cd ~: will take you to the home directory
  • cd .. :moves up one directory
  • ls :lists the files or directories
  • git init: Creates a new local repository in the current directory
  • git clone: Clones a repository from a remote resource (GitHub, Bitbucket, etc.) Creates a directory inside the current directory, and puts the project in there
  • git checkout -b {branch name}: shorthand command to create and switch to a new branch
  • git branch {branch name}: creates a new branch
  • git checkout {branch name}: switches to the branch
  • git branch -a: displays all branches, both local and remote, that are available
