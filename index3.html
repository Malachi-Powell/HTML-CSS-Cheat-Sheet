  • git status: Tells what files have been added, deleted, and modified Informs if the local branch is behind the remote version or if the version is ahead of the remote repository

  • git push: Takes the locally committed changes for a given branch and applies them to the local version

  • git push -u origin {branch name}:This command is used for pushing to the remote repository with the specific branch name

  • git pull: Takes the changes made to the remote repository and applies them to the local version
